# Script-Debian / Post-install
Ceci est un script pour l'installation de paquets sur un VM Linux.

## Pré-requis
Avoir un machine sous **Debian 13.2.0** ou **Ubuntu 24.04**.

## Contenu
Ce script comprend l'installation des paquets : `ssh`, `zip`, `namp`, `locate`, `ncdu`, `curl`, `git`, `screen`, `dnsutils`, `net-tools`, `sudo`, `lynx` et `tree`.
